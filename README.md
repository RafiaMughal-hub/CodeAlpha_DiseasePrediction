# CodeAlpha_DiseasePrediction
# Heart Disease Prediction Using Machine Learning Algorithms

## Project Overview

This project predicts the likelihood of heart disease using machine learning algorithms. Patient health data is analyzed to classify whether a patient is at risk of heart disease.

## Objective

To build and evaluate machine learning models that can accurately predict heart disease using medical attributes.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Dataset

Heart Disease Dataset

Features include:

* Age
* Sex
* Chest Pain Type
* Cholesterol
* Resting Blood Pressure
* Maximum Heart Rate
* Exercise Induced Angina

## Machine Learning Models

1. Logistic Regression
2. Random Forest Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix
* ROC Curve
* ROC-AUC Score

## Results

Random Forest achieved the best performance and produced a strong ROC-AUC score, demonstrating reliable prediction capability.
